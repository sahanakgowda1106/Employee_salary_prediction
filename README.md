# 📊 Employee Salary Prediction

## 📌 Project Overview
This project predicts employee salaries based on key attributes such as experience, education level, skill score, and certifications.

---

## 🎯 Objective
- Analyze employee data  
- Build a model to predict salary  
- Evaluate prediction accuracy  

---

## 📂 Dataset
**File:** `employee_salary_prediction.csv`

### Features:
- `years_experience`  
- `education_level`  
- `skill_score`  
- `certifications`  

### Target:
- `salary`  

---
